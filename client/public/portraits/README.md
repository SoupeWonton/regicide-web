# Hero portraits

Drop one square image per class here and the battlefield picks it up
automatically (missing files fall back to the class emblem):

```
sentinel.png   quartermaster.png   surgeon.png   executioner.png
commander.png  warden.png          gambler.png   exile.png   oracle.png
```

Recommended: 256×256 PNG, dark painterly fantasy bust on a dark background
(they render in a ~50px rounded frame). AI-generate them with a prompt like:

> "Dark fantasy oil painting bust portrait of a medieval SENTINEL knight with
> tower shield, dramatic candlelight, deep indigo and tarnished gold palette,
> dark background, Slay the Spire character art style"

…swapping the class flavor per file: quartermaster (supply officer, ledger and
satchel), surgeon (plague-doctor field medic), executioner (hooded axeman),
commander (banner officer), warden (lantern keeper), gambler (sly dice-roller),
exile (burned outcast), oracle (blind seer).
