# Sim run 2026-06-10 — 1,150 campaigns, 6,174 encounters

Generated by `server/scripts/sim.ts` (`npx tsx scripts/sim.ts --seeds 50`).
50 seeds × player counts 1-4 × lineups (slayer / bulwark / hoarder / sniper / steady / mixed).
Personas are weight profiles over the same bot logic: slayer = max damage, bulwark = shields/safety,
hoarder = card economy, sniper = exact kills, steady = control, mixed = one of each.

## Files

- `runs.csv` — one row per campaign: result, chapter reached, deaths, retreats, funnel flags
  (reachedBoss1 / beatCh1 / reachedBoss2), per-persona death tally.
- `encounters.csv` — one row per fight: tier, modifier, outcome, deaths, turns, attempt #.
- `summary.json` — pre-aggregated tables (by count, lineup, tier, modifier, boss modifier).

## Headline findings

- **Ch1 boss is the wall**: 12.4% win rate, 867 of 1,011 losses happen there. Road content is
  fine by comparison (skirmish 95% / veteran 74% / elite 55% won). Ch2 boss identical (12.1%).
- **Economy beats aggression**: funnel reached-boss → beat-ch1 → won is 91.5% → 18% → 3.5% for
  hoarder vs 76.5% → 3.5% → 0.5% for slayer. In mixed parties the slayer slot dies 3.3× more
  than the sniper slot (213 vs 65 deaths over 150 runs).
- **Player count**: solo never won (boss is win-or-wipe), 2p 0.3%, 3p 2.0%, 4p 1.3%. The F1-F3
  low-count scaling helps the road but not the castle.
- **Modifier spread is wide**: banner-of-knives 49% won / 0.82 deaths per fight vs a bottom six
  (fog-marker, dry-cart, bleeder-patrol, hooked-blades, cracked-buckler, wrong-relay) at 93-96%
  won / ~0.07 deaths. starved-caravan forces the most retreats (152 in 489 fights).
- **starving-court** is the hardest ch2 boss modifier by far (6.7% vs ~15% for iron/cruel court),
  confirming prediction E5 in MECHANIC-AND-BALANCE-ISSUES.txt.

Bots are decent but sub-human — read absolute win rates as a floor; relative comparisons
(persona vs persona, modifier vs modifier, count vs count) are the trustworthy signal.
