# Items

Relics are very rare, slotless exceptions—not a managed inventory or the class
progression tree. A keep-or-sacrifice decision should occur at most rarely in a run.

Standalone spell inventory and spell currency are removed from V3. Burst effects
belong on cards or existing combat actions rather than in a parallel item system.
Exact relic content remains to be specified.
