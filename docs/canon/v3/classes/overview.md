# Classes

A class is a starting hand plus one innate loophole in the combat loop. The loophole
deepens through the campaign without a separate point economy or relic-slot tree.

The initial roster is deliberately four sharply separated roles:

- Sentinel: block and defense.
- Executioner: kill and aggression.
- Quartermaster: combine and tempo.
- Surgeon: persist and recovery.

Exact kits and the progression model remain proposals. The linear model lives in
[`four-core-classes.md`](../../../proposals/classes/four-core-classes.md); the competing
facet/permutation exploration lives in
[`facets-and-pressure-permutations.md`](../../../proposals/classes/facets-and-pressure-permutations.md).
The other five V2 classes are future unlock runway, not active V3 specifications.
