# V3.0 build decisions — closing every §Q blocker in one sweep

Purpose: answer or delegate **all** remaining open product decisions in
[`../delivery/plans/v3.0-integration.md`](../delivery/plans/v3.0-integration.md) §Q so the
nine build slices can be implemented **autonomously** (no further Landry input) until
human playtest. Numbers remain placeholders per plan §A.

## Decisions (Landry, 2026-07-01)

| # | §Q | Decision |
|---|---|---|
| 1 | Q3 | **Starting decks: identical 20-card A–5 × 4 suits for all four classes** (matches live). Class identity = Staff choice + home-suit path only. No skew, no pre-applied grafts. |
| 2 | Q6 | **Cast = consume to empty.** Casting a suit's spell empties that hole entirely — all sandbagged progress spent. Tension = cast now vs build toward Half. |
| 3 | Q12/Q28 | **Graft only — no fragment on the graft trigger.** Fragments come solely from the post-encounter 50/50 roll. Two separate systems. |
| 4 | Q16/Q8 | **Auto seam reset + optional Camp, distinct powers.** Every province seam = automatic reshuffle + redraw (in the recap screen), **no block, no double-attack**. Camp stays an optional road landmark with a **four-part bundle**: reshuffle discard → Tavern · **draw up to 5** · first attack doubles · **start the next fight with 10 block** *(block addition new this decision — canon)*. |
| 5 | Q7 | **C2 provinces mirror the C1 skeleton** — reuse the live chapter road shape (fights + forks + ~1 landmark stop + boss); only the boss changes: P1 → Jack Gate, P2 → Queen Gate (Fallen Heroes at P2 start), P3 → King Gate. |
| 6 | Q10 | **"Redraw to 5" = draw up to 5** — keep the current hand, draw until holding 5 (a top-up, not a discard-and-redraw). Applies to Camp and the seam reset. |
| 7 | Q11 | **Relic swaps: free at every between-encounter screen.** No cost, no landmark gating; add friction later only if playtest demands. (Implicitly: relics never change mid-fight.) |
| 8 | Q11 | **Fragments: uncapped bracelet pool.** Unplaced fragments bank without limit; place any time between encounters. Suits cap at **Half** in V3.0 (progress toward Full banks silently for V3.5). Nothing is ever lost. |
| 9 | Q4 | **Staff/path semantics: engineering pins all 32** (16 Staffs + 16 ladder rungs) per the survivor-test principles, each recorded in a contracts doc; ambiguous readings flagged there. Landry reviews at playtest, not before. |
| 10 | Q5 | **Relic contracts: engineering pins all 29** (trigger/timing/edge cases, placeholder numbers — e.g. Sainted Scalpel N=6 from live), fragment-touching relics simplified to agnostic fragments. Same contracts-doc treatment. |
| 11 | Q14 | **UI: functional placeholder UI for all 8 screens**, built in the existing client style. Gab restyles later; mechanics won't change. |
| 12 | scope | **Build all 9 slices autonomously in order** (§F → grafts → gates → classes → forgiveness → spells → equipment → landmarks → provinces/meta), smoke tests green after each slice, stopping only when V3.0 is playable end-to-end (C1+C2, solo, victory = C2 clear). |

Q1 (card-state model) is engineering-confirmed by the same sweep; Q2 (live audit) closed
2026-07-01 ([`../delivery/live-code-audit-91d3677.md`](../delivery/live-code-audit-91d3677.md));
Q13 (lineage schema) is delegated to engineering within the decided shape (wipe on cutover;
C2-clear unlocks all paths; meta = options not power).
