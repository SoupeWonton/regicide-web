# Landmarks

**Summary:** Landmarks are short route interruptions with one recognizable verb, such as resting, rearranging grafts, pursuing a recruit, or risking combat for a relic.

A consolidated map now exists at [[proposals/systems/landmarks|the landmarks proposal]]: each landmark is **one verb, one telegraphed decision** — Sanctum *attune* · Lair *raid* · Caravan *pay-from-hand* · Camp *prepare* · Forge *rearrange* · Hunt *pursue* · **Shrine *consecrate*** · Tower *removed*. Every surviving location must change a route decision without becoming a generic shop. (sources: [[proposals/systems/landmarks|Landmarks proposal]], [[canon/principles/content-bar|Content bar]], [[proposals/open-design-questions|Active design questions]])
