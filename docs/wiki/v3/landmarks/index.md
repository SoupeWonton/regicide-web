# Landmarks

**Summary:** Landmarks are short route interruptions with one recognizable verb — resting,
rearranging grafts, pursuing a recruit, risking combat for a relic. Each is **one verb, one
telegraphed decision**, never a generic shop. The consolidated map lives at
[[proposals/systems/landmarks|the landmarks proposal]].
