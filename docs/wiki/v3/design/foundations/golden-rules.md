# Golden rules

**Summary:** Eleven accepted tests protect core fun, tradeoffs, discovery, story, synergy, divergence, and contextual card meaning.

The core loop must remain fun without progression; runs should begin with meaningful choices; and power should require sacrifice or constraint. Builds should emerge through play rather than be fully selected in advance. (source: [[canon/principles/golden-rules|Golden rules]])

Runs should create memorable stories, support synergy at several scales, prefer mechanical changes over numerical inflation, and diverge visibly. Rewards must continue to create decisions, while repetition reveals new possibilities instead of repeating an identical experience. (source: [[canon/principles/golden-rules|Golden rules]])

Cards gain meaning through deck, relic, class, and run context. The same card should not have one fixed value across every build. (source: [[canon/principles/golden-rules|Golden rules]])

| Rule | Practical reading inside V3 |
|---|---|
| Core loop first | [[v3/mechanics/core-loop|Combat]] must remain worth playing without unlocks masking it. |
| Meaningful start | [[v3/classes/class-identity|Class and starting state]] should distinguish runs before combat. |
| Tradeoffs | Grafts, routes, relics, and recovery should surrender something real. |
| Discovered builds | The conquered deck should redirect a plan rather than merely execute a lobby choice. |
| Run stories | Players should recount sequences and recoveries, not percentage totals. |
| Multi-scale synergy | Small card interactions, build engines, and surprising combinations should coexist. |
| Mechanical over numerical | Prefer changed behavior to flat bonuses. |
| Divergent runs | Two victories should be visibly and strategically different. |
| Continuing decisions | Rewards should resist obvious autopicks. |
| Discovery through repetition | Meta breadth should reveal new plans rather than grind permanent stats. |
| Contextual cards | A card's value should change with class, grafts, pressure, and deck state. |

The rules intentionally pull against each other. More relic interaction may improve synergy and discovery while increasing [[v3/design/foundations/depth-width-and-elegance|width]]; stronger starting identity may improve immediate choice while predetermining the build. They are evaluation lenses, not independent scores to maximize, so a healthy decision explains which rule it advances and which cost it accepts. (sources: [[canon/principles/golden-rules|Golden rules]], [[research/assessments/golden-rules-2026-06-18|Golden Rules assessment]])

The raw playtest recovery story is a useful example of several rules aligning: persistent scarcity created tension, a sequence of draws and recovery revealed a line, and the tester narrated the chain afterward. That evidence does not validate the current balance, but it shows what “story,” “context,” and “discovery” sound like in player language. (sources: [[research/playtests/raw/Updated_Raw.txt|Updated raw feedback]], [[archives/v2/playtests/raw/Final_Update.txt|Final raw feedback]])
