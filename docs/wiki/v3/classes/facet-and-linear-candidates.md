# Class progression — facet & linear candidate pool

**Summary:** An over-generated pool of class-progression candidates for the Q5 decision —
multiple **linear ladders** and **facet sets** for all four classes — built for an
elimination ("KEEP/CUT") pass. Names/numbers are illustrative.

Full pool, rationale, scaling/playstyle notes, and the worksheet:
[[proposals/classes/facet-and-linear-candidates|candidate pool (source)]].

## The fork

- **Linear** — the class *is* one fixed ladder (e.g. Sentinel: Aegis→Bulwark→Impenetrable).
  Simpler to build/teach; replay variety comes only from deck/relics/route.
- **Facet** — the class is a *verb*; pick one mutually-exclusive dialect ~Act 2 (Sentinel:
  Bastion / Riposte / Protector). Different characters per run; meta unlocks add facets;
  ~3× content to balance.
- **Hybrid** (flagged in the source) — fixed tier-1 for instant legibility **+** a facet for
  tiers 2–3.

> Facets express "**breadth via choice, not vertical power**" — the same principle as the
> 4-slot spell tiers and unique relics. If those are accepted, facets rhyme with them.
