# Classes

**Summary:** V3 begins with four classes, each defined by a starting hand and one innate exception to a different station of the card loop.

Sentinel owns Block, Executioner owns Kill, Quartermaster owns Combine, and Surgeon owns Persist. Those identities are accepted direction; exact kits and progression remain competing proposals.
