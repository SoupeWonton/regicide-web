# Doorstop

**Summary:** Doorstop is a proposed exact-block relic that returns the Spade card which reduces an enemy's attack precisely to zero.

**Doorstop** is a proposed defensive relic: once per enemy, when a Spade card reduces that enemy's attack to exactly 0, return that Spade to the player's hand. The card that completes the wall visibly bounces back. (source: [[proposals/systems/relics-and-spell-cards|Relics and spell cards proposal]])

This creates an “exact block” parallel to exact killing. The once-per-enemy boundary prevents repeatedly refunding the same card after attack has already reached 0, while the returned card preserves future offense, defense, or discard value. (source: [[proposals/systems/relics-and-spell-cards|Relics and spell cards proposal]])

The V2 Doorstop instead carried shield between enemies; that older rule inspired the defensive identity but is not the current candidate text. Timing, multi-card Spade plays, interaction with the rare [[v3/spells/proposed-design/spades-spell|Spades spell]], and Sentinel amplification remain open. (sources: [[archive/v2/items/items-v2|V2 item rebuild]], [[proposals/systems/relics-and-spell-cards|Relics and spell cards proposal]])
