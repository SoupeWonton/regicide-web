# Amulet relics — activated *(proposed candidates)*

**Slot theme (locked 2026-06-28):** **Amulet = activated** — a button you press; it may touch any
lever but is gated behind an activation. The key distinction from a **spell**: a spell is **one-use
and consumed**; an Amulet **refreshes** (once per fight / per enemy). (source: [[proposals/systems/relic-candidates-by-slot|Relic candidates by slot]])

**Review pass 1 (Landry, 2026-06-28):** applied below — keepers refined, *Last Breath* cut.

> **✅ Officialized 2026-06-28 → [[canon/v3/systems/relics|`relic_v1_design_3.0`]].** In the pool:
> Sainted Scalpel · Unbinding · Second Wind · Aegis · Bloodlust · Echo · Lodestone. **Held** (still
> `proposed`): **Transmute** (still strong) · **Ebb** (needs spec).

**Legend:** ⚠ still possibly too strong (watch) · 👁 info-value · 🔁 brushes a class lane · ★ kept reference · ✦ needs exact spec.

## Active candidates (reviewed)

| Relic | The one rule it bends *(Activate:)* | Status / note |
|---|---|---|
| **Sainted Scalpel** ★ | *Once per fight:* shuffle up to N cards from discard into the Tavern and draw 1. | kept recover benchmark. |
| **Unbinding** (Jester-stone) | *Once per enemy:* cancel the enemy's immunity **for this play only** (not the rest of the fight). | tamed from whole-fight → single action. |
| **Transmute** | *Once per encounter:* raise your combo cap from 10 to 14 for one play. | ⚠ rarer (encounter, not enemy); may still need a nerf. |
| **Second Wind** | *Once per fight:* take an extra turn before the enemy counterattacks. | ✓ liked — reads almost like a soft attack-cancel. |
| **Aegis** | *Once per enemy:* reduce the enemy's next counterattack by 5. | "a bit boring" — flagged to revisit (full-negate was too strong). |
| **Bloodlust** | *Once per enemy:* your next play deals **+3 damage** (not double). | ✓ toned from double → +3. |
| **Echo** | *Once per fight:* replay a card from your discard for **its value only — its suit power does not trigger**. | ✓ liked; value-only is the rule. |
| **Ebb** | *Once per enemy:* reduce the enemy's attack to 0 this turn by paying half its value (round down) in cards. | ✦ needs exact spec — timing, and what "paying half in cards" resolves to. |
| **Lodestone** | *Once per fight:* pull one specific named card from the Tavern/library into your hand. | ✓✓ loved — the tutor. |

> **Migration note:** **Debt** (currently under [[v3/relics/proposed-design/ring-economy-candidates|Ring]])
> is also a candidate for this slot — it's an activated effect.

## Cut

- **Last Breath** — death-save; off-design (risks erasing attrition, the parked *Iron Reprieve* danger).
