# Relics — philosophy & design orientation

**Status:** Lead doc for V3 relics. Grounded in the
[[decisions/2026-06-24-crystals-continents-and-equipment|crystals, continents & equipment decision]]
(Decision C), which supersedes the slotless model.

## What a relic is

A relic is a **rare, run-defining exception** held as **equipment** in one of four slots
(**Cloak · Ring · Hat · Amulet**, alongside the class **Staff**) — *not* the class ladder, *not* a
stat boost. Each relic **bends exactly one rule of the core loop** in a legible, narratable way
(e.g. *Split Seal*: a graft may target any owned card; *Doorstop*: an exact block bounces the Spade
back to hand; *Crown of First Claim*: a fresh recruit enters hand immediately). Slot themes
(Cloak ≈ roads, Ring ≈ economy, Hat ≈ recruitment, Amulet ≈ activated) are placeholders, not fixed.

## The bar every relic must clear

- **One rule, bent once** — a relic changes a single rule of `draw → combine → kill → block →
  persist`, statable in one sentence. If it needs a paragraph or a subsystem, it's not a relic.
- **Story, not stats** — it must *tell a story* and **not duplicate a class loophole or a graft**.
  A relic that merely amplifies a class identity needs scrutiny: does it erase what makes that class
  unique when a *different* class picks it up?
- **Rare, four slots** — most runs see **one or two** relics across the Cloak/Ring/Hat/Amulet
  slots. Breadth comes over a lifetime (death / milestones widen the pool), never vertical power
  within a run.
- **Earned, not handed out** — **primary source: the Lair** (verb *raid* — win it through a
  dangerous fight); **costly alternative: the Caravan** (pay from hand). Never dropped at ordinary stops.
- **Staffs swap at Fallen Heroes** — the class-ability Staff is the fifth slot; relics fill the other four.
- **Power baseline is a human-playtest question**, never a sim number.

## Orientation for this pass

For each relic, pressure-test:
1. **The one rule** — name the single loop rule it bends, in one sentence.
2. **Non-duplication** — does it overlap a class loophole, a graft, a spell, or another relic? If so,
   differentiate or cut.
3. **Narratability** — can a player explain what it does *and why it's cool* in a breath?
