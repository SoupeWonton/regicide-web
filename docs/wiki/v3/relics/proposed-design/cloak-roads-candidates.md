# Cloak relics — roads *(proposed candidates)*

**Slot theme (locked 2026-06-28):** **Cloak = roads** — the between-fight / route / persistence layer.
The cleanest relic slot: every class lives *inside* a fight, so the road layer is design space no
class touches. (source: [[proposals/systems/relic-candidates-by-slot|Relic candidates by slot]])

**Review pass 1 (Landry, 2026-06-28):** six keepers below; *Waystone* parked; *Momentum / Long Shadow /
Funeral March* cut. **Rule clarified: losing a fight = death** (no flee-with-reward).

> **✅ Officialized 2026-06-28 → [[canon/v3/systems/relics|`relic_v1_design_3.0`]].** All six keepers
> are in the pool. **Held:** *Waystone* (parked).

**Legend:** 👁 info-value · 🩹 floor-relief.

## Active candidates (reviewed)

| Relic | The one rule it bends | Status / note |
|---|---|---|
| **Forked Road** | At a road fork, you see and choose between *both* branches' landmark/enemy type before committing. | ✓ liked. 👁 route info, not deck-scry. |
| **Forced March** | You may skip one ordinary fight per road (slip past) — but you don't recruit or graft from it. | ✓ liked. |
| **Bedroll** | Once per road, reshuffle your discard into the Tavern *without* a Camp (a pocket rest). | ✓ liked. |
| **Vanguard** | The first enemy of each new road cannot counterattack on its first turn — a free opening turn. | ✓ liked. |
| **Slip Away** | **Discard 5** to retreat from a fight: keep your hand, but the enemy is not defeated. | ✓ liked; toll = discard 5. |
| **Scout Ahead** | You see the *immunity* (locked lever) of the next enemy on the road before you fight it. | ✓ liked. 👁 actionable here. |

## Parked (good idea, too strong for now)

- **Waystone** — bank a second landmark visit on a road (double-stop, then skip ahead). Liked, but too
  strong right now; kept in the idea pool.

## Cut

- **Momentum** — a stacking first-attack bonus. Cut: **no stacking / alternate economy.**
- **Long Shadow** — first-card immunity pierce. Cut: not actually about the *road* layer (off-theme).
- **Funeral March** — graft-on-loss. Cut: **losing a fight = death**, so there's no flee-with-reward.
