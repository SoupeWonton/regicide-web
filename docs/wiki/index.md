# Kingfall encyclopedia

**Summary:** This Obsidian encyclopedia organizes current V3 knowledge by mechanics, landmarks, spells, relics, classes, stages, and design, while preserving V2 and earlier material in a separate searchable archive.

The wiki is a derived navigation and explanation layer. Source documents remain authoritative: code and tests describe shipped behavior, canon describes intended V3 behavior, decisions explain accepted changes, delivery records implementation gaps, proposals remain unsettled, and archives remain historical. (sources: [Documentation index](../README.md), [[canon/README|Canon index]])

### [[v3/mechanics/index|Mechanics]]

Combat, exact kills, conquest, replacement grafts, persistent deck state, lifecycle, items, and the shared card engine.

### [[v3/landmarks/index|Landmarks]]

### [[v3/stages/index|Stages]]

The continuous **five-continent** expedition (Claim→Master, God of Luck overlay, opt-in ending), with nested [[v3/stages/continents/index|continents]] and [[v3/stages/continents/continent-1/provinces/index|province]] navigation ready for accepted content.
