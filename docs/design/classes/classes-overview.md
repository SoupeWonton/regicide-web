# Classes

v0 class roster and loadout rules.

## Core Rules

- One player controls one hero.
- Campaign start uses core heroes only.
- Specializations unlock after Chapter 1 completion.
- Chapter 1 starts without specializations.

## Tier Model

### Tier 1: Core Heroes

Starter roster:
- Sentinel
- Quartermaster
- Surgeon
- Executioner

Role: baseline playable foundation for all parties.

### Tier 2: Support/Advanced Heroes

Current roster:
- Commander
- Warden

Role: unlock through progression and appear through replacement/camp recruitment flow.

### Tier 3: Weird/System Heroes

Current roster:
- Gambler
- Exile
- Oracle

Role: advanced unlocks with stronger system-level interactions and higher execution demand.

## Replacement Rules

- Tier 2 and Tier 3 may enter mid-campaign via death flow and camp recruitment.
- Replacement class must differ from class that died.
- Immediate join is allowed with reduced onboarding bonuses.
- Camp join is allowed with stronger onboarding bonuses.

## Progression Rules

Kingdom progression can unlock:
- classes
- chapter access
- specialization pathways

Unlock persistence is permanent at Kingdom level.

## Party Size Notes

- Existing multiplayer supports 1-4 players.
- Four players is preferred full-party baseline.
- Lower player counts still follow one-player-one-hero control.
