## The bar

- **A passive is an ENABLER, not a payoff** — it changes *how often / how easily* the key moment
  fires (its timing, scope, cost, reachability), and **never grants a scaling resource.**
- **A whisper, costed** — fires once per enemy or relaxes one rule; concrete value, small.
- **Never** information/scry, never a free resource, never another suit's job.
- Each class has **four** Staffs (16 total); a Staff pairs with any class (decoupled from the path).
