## The bar

- **One verb, one decision** — never a generic shop or a reward buffet.
- Must answer: *why would this deck route here now, and what competing opportunity is surrendered?*
- Acquisition stays capped (~3 item-stops per act) so landmarks punctuate combat, not replace it.
- A new landmark must not collide with an existing verb (Forge=spells, Sanctum=deck-mod, Lair=relic, …).
