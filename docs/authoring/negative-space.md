# Negative space — do NOT propose these

The patterns ruled out, with the why. Check a draft against this list before proposing.

- **No secondary wallet / generic currency.** The one bounded exception is spell **fragments** — now
  **agnostic** tokens (50/50 drop after each encounter) armed on the **bracelet** to set the next
  encounter's spell (revised 2026-06-28). They stay **single-purpose** (arm spells), never a general
  wallet. See [`constraints`](../canon/v3/constraints.md) — confirm its "suit-specific, Forge-only"
  wording still matches.
- **No card-removal faucet.** Deck thinning is unexplored, not a tool to hand out.
- **No scry / foresight value.** The deck churns 3–4× per boss, so "see the top N" is worthless.
- **No "all enemies" effects** — fights are one royal at a time.
- **Curses removed.** Not a V3 mechanic.
- **No mid-run save/resume.** A run is single-session; only lineage meta persists.
- **Retired:** siege ultimates · the `tokenBudget` (forge/graft budget) · additive Hone (+1) and
  Graft (add-suit) — grafts are **replacement-only** · the Tower landmark.
